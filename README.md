## Analyses for "Dosage of the fusion transcription factor PAX3::FOXO1 controls cell state in rhabdomyosarcoma"

The scripts used for each main figure and the corresponding supplementary figures/tables are contained within separate directories as delineated below: 
* **figure_1**: Figure 1, Supplemental Figure 1
* **figure_2**: Figure 2, Supplemental Figures 2, 3, 4, 5, and 6
* **figure_3**: Figure 3, Supplemental Figure 7, Supplemental Table 1
* **figure_4**: Figure 4, Supplemental Figures 8 and 9
* **figure_5**: Figure 5, Supplemental Figure 14, Supplemental Table 3
* **figure_6**: Figure 6, Supplemental Figure 15
* **revision_figures**: Supplemental Figures 10, 11, 12, 13

Comments indicate the figure panels created by each script. Note that Supplemental Tables 2 & 4 are not listed because they contain data from across multiple figures/supplemental figures. The initial bioRxiv version of this work is located here: https://www.biorxiv.org/content/10.1101/2025.05.16.654522v1.
